Overview

This command is used to retrieve information about the encoding profiles saved into the MMS repository.

URI

Encoding profiles

  • catramms/v1/profile/[video | audio | image]/[:encodingProfileKey]

Encoding profiles Set

  • catramms/v1/profilesSet/[video | audio | image]/[:encodingProfilesSetKey]

Method

GET

URL Params

Required

None

Optional

  • encodingProfileKey=[number]
  • example: encodingProfileKey=123
  • encodingProfilesSetKey=[number]
  • example: encodingProfilesSetKey=123

Payload Data

None

Success Response

Code

200 OK
