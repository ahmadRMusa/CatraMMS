Overview

This command is used to retrieve information about the contents saved into the MMS repository.

URI

catramms/v1/mediaItems/[:mediaItemKey]/?[:start]&[:rows]&[:contentType]&[:startIngestionDate]&[:endIngestionDate]

Method

POST

URL Params

Required

None

Optional

  • mediaItemKey=[number]
  • example: mediaItemKey=123
  • start=[number starting from 0]
  • default: 0
  • example: start=20
  • rows=[number]
  • default: 10
  • example: rows=10
  • contentType=[string: video, audio or image]
  • example: contentType=video
  • startIngestionDate=[string, format: YYYY-MM-DDTHH:MI:SSZ]
  • example: startIngestionDate=2018-04-26T07:19:11Z
  • endIngestionDate=[string, format: YYYY-MM-DDTHH:MI:SSZ]
  • example: endIngestionDate=2018-04-27T07:19:11Z

Payload Data

See the JSON format of the command it is ingesting
