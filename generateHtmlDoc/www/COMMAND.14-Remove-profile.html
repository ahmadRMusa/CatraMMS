Overview

This command is used to remove one 'Profile' or a set of profiles from the CatraMMS.

URI

  • to remove an encoding profile (video, audio or image)
  • /catramms/v1/profile/:encodingProfileKey
  • to remove an encoding profiles set (video, audio or image)
  • /catramms/v1/profilesSet/:encodingProfilesSetKey

Method

DELETE

URL Params

Required

  • encodingProfileKey: encoding profile key to be removed
  • Example: 456
  • encodingProfilesSetKey: encoding profile set key to be removed
  • Example: 765

Optional

None

Payload Data

None

Success Response

Code

200 OK

Content

None

Error Response

See Common Error Responses

Sample Call

Assuming: * UserKey: 5 * APIKey: SU1.8ZO1O2zVeBMNv9lzZ0whABXSAdjWrR~rpcnI5eaHNO2gysrNdAv8BNoWZB1eLP

curl -k -v -X DELETE -u 5:SU1.8ZO1O2zVeBMNv9lzZ0whABXSAdjWrR~rpcnI5eaHNO2gysrNdAv8BNoWZB1eLP https://mms-api.catrasoft.cloud/catramms/v1/profile/456
