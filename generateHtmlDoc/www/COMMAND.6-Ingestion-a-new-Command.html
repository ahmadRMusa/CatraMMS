Overview

This command is used to execute any kind of ingestion into the CatraMMS.

URI

/catramms/v1/ingestion

Method

POST

URL Params

Required

None

Optional

None

Payload Data

See the JSON format of the command it is ingesting

Success Response

Code

201 Created

Content

The API returns the identifiers (Keys) and labels about the 'workflow' and every single task just added in the ingestion queue.

{ 
    "workflow": { "ingestionRootKey": 8, "label": "..." }, 
    "tasks": [
        { "ingestionJobKey": 16, "label": "..." }, 
        { "ingestionJobKey": 17, "label": "..." }, 
        { "ingestionJobKey": 18, "label": "..." },
        ... 
    ]
}

Error Response

See Common Error Responses

Sample Call

Assuming:

  • UserKey: 5
  • APIKey: SU1.8ZO1O2zVeBMNv9lzZ0whABXSAdjWrR~rpcnI5eaHNO2gysrNdAv8BNoWZB1eLP
  • contentMetadata.json is a file containing the metadata about the Workflow and related tasks using the language.

curl -k -v -X POST -u 5:SU1.8ZO1O2zVeBMNv9lzZ0whABXSAdjWrR~rpcnI5eaHNO2gysrNdAv8BNoWZB1eLP -d @contentMetadata.json -H "Content-Type: application/json" https://mms-api.catrasoft.cloud/catramms/v1/ingestion
