Overview

This command is used to add ome 'Profile' into the CatraMMS. In case the profile is already present, this one will be updated.

URI

  • to add a video profile
  • /catramms/v1/profile/video
  • to add an audio profile
  • /catramms/v1/profile/audio
  • to add an image profile
  • /catramms/v1/profile/image

Method

PUT

URL Params

Required

None

Optional

None

Payload Data

See the JSON format for the appropriate 'Profile'

Success Response

Code

201 Created ### Content The API returns the identifier (Key) and name of the profile.

{
   "encodingProfileKey": 16,
   "label": "..." 
}

Error Response

See Common Error Responses

Sample Call

Assuming: * UserKey: 5 * APIKey: SU1.8ZO1O2zVeBMNv9lzZ0whABXSAdjWrR~rpcnI5eaHNO2gysrNdAv8BNoWZB1eLP * profileMetadata.json is a file containing the video metadata about the 'Profile'

curl -k -v -X PUT -u 5:SU1.8ZO1O2zVeBMNv9lzZ0whABXSAdjWrR~rpcnI5eaHNO2gysrNdAv8BNoWZB1eLP -d @profileMetadata.json -H "Content-Type: application/json" https://mms-api.catrasoft.cloud/catramms/v1/profile/video
