Overview

This command is used to retrieve information about an ingestion.

URI

/catramms/v1/ingestion/:IngestionRootKey/[:IngestionJobKey]

Method

GET

URL Params

Required

  • IngestionRootKey=[number] example: IngestionRootKey=12

Optional

  • IngestionJobKey=[number] example: IngestionJobKey=543

Payload Data

None

Success Response

Code

200 OK

Content
