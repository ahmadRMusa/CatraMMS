Pull option

The Pull option means the binary of the content is downloaded by the MMS. The protocols supported are: http, https, ftp and ftps. The pull option supports the Resume in case of a fault during the download of the content.

  • In the Pull option the metadata contains the URL to be used to download the content ('SourceURL' item)
  • A resume might be used in case the download fails for any reason. Moreover a number of retries is used in case of network issues.
