Overview

  • Perform a Login and returns, in case of success, all the couples WorkspaceName and APIKey the User is able to work on.

URI

/catramms/v1/login

Method

POST

URL Params

Required

None

Optional

None

Payload Data

{
   "EMail": "admin@mycompany.com",
   "Password": "kskhkd_3rR"
}

Success Response

Code

200 OK

Content

{ 
   "userKey": 198,
   "workspaces": [ 
      { 
         "workspaceName": "CatraSoft", 
         "apiKey": "SU1.8ZO1ODJDHFOFOIOOFJJSrQ_S4lS33iZ4zFJ8kfWxQqmOhkL", "owner": 1 
      } 
   ] 
}

Error Response

See Common Error Responses

Sample Call

curl -k -v -X POST -d '{"EMail": "giuliano@catrasoftware.it", "Password": "kjkdiif" }' https://mms-api.catrasoft.cloud/catramms/v1/login
