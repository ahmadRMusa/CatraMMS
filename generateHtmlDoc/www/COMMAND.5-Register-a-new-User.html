Overview

  • Create a new User and his relative Workspace (disabled) in the MMS
  • An email is sent to the User containing the confirmation code. It will be used to confirm the User and enable the Workspace (confirm).
  • The userKey is returned to the client

URI

/catramms/v1/user

Method

POST

URL Params

Required

None

Optional

None

Payload Data

{
   "Name": "giuliano",
   "EMail": "admin@mycompany.com",
   "Password": "kskhkd_3rR",
   "Country": "Italy",
   "WorkspaceName": "MyWorkspace"
}

Success Response

Code

201 Created

Content

{ 
   "userKey": 12
}

Error Response

See Common Error Responses

Sample Call

curl -k -v -X POST -d '{"Name": "Giuliano", "EMail": "giuliano@catrasoftware.it", "Password": "giuliano", "Country": "Italy", "WorkspaceName": "CatraSoft" }' https://mms-API.html.catrasoft.cloud/catramms/v1/user
