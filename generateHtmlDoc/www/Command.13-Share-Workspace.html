Overview

  • Share a Workspace with another User
  • Create a new User if not already present in the MMS
  • An email is sent to the User containing the confirmation code. It will be used to confirm that the User is able to work with the shared Workspace (confirm).
  • The userKey is returned to the client

URI

/catramms/v1/workspace/:WorkspaceKey/share?:ingestWorkflow&:createProfiles&:deliveryAuthorization&:shareWorkspace

Method

PUT

URL Params

Required

  • WorkspaceKey=[number]
  • example: WorkspaceKey=12
  • ingestWorkflow=[boolean]
  • example: ingestWorkflow=true
  • createProfiles=[boolean]
  • example: createProfiles=true
  • deliveryAuthorization=[boolean]
  • example: deliveryAuthorization=true
  • shareWorkspace=[boolean]
  • example: shareWorkspace=true

Optional

None

Payload Data

{
   "Name": "giuliano",
   "EMail": "admin@mycompany.com",
   "Password": "kskhkd_3rR",
   "Country": "Italy"
}

Success Response

Code

201 Created

Content

{ 
   "userKey": 12
}

Error Response

See Common Error Responses

Sample Call

curl -k -v -X POST -d '{"Name": "Giuliano", "EMail": "giuliano@catrasoftware.it", "Password": "giuliano", "Country": "Italy" }' https://mms-api.catrasoft.cloud/catramms/v1/workspace/123/share?ingestWorkflow=true&createProfiles=false&deliveryAuthorization=false&shareWorkspace=false
