Overview

  • A User already registered has to be ‘confirmed’ through the confirmation code received by the User through an email
  • Return the API.html Key to be used to call the MMS API.htmls

URI

/catramms/v1/user/:UserKey/:ConfirmationCode

Method

PUT

URL Params

Required

  • UserKey=[number] example: UserKey=12
  • ConfirmationCode=[string] example: ConfirmationCode=1387270786

Optional

None

Payload Data

None

Success Response

Code

200 OK

Content

{ 
   "API.htmlKey": "SU1.8ZO1O2zRs.gL_nWYV4AZ0uU_dy89CXogV2eLaeQ0"
}

Error Response

See Common Error Responses

Sample Call

Assuming:

  • UserKey to be confirmed: 2
  • Confirmation Code of the User to be confirmed: 2049792014

curl -k -v -X PUT https://mms-API.html.catrasoft.cloud/catramms/v1/user/2/2049792014
